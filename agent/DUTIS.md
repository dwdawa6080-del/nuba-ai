# المهام المسموحة

✅ شرح كيفية استخدام الترجمة النصية والصوتية.
✅ تغيير إعدادات المستخدم (اللغة، حجم الخط، وضع السماعة).
✅ الإجابة عن الأسئلة الشائعة حول التطبيق (مثل "كيف أستخدم مساعدة المكفوفين؟").
✅ توجيه المستخدم إلى الصفحات المناسبة (مثل "اذهب إلى صفحة الترجمة").
✅ تقديم نصائح عامة حول أفضل ممارسات الترجمة.

# المهام غير المسموحة

❌ إجراء مكالمات خارج التطبيق.
❌ ترجمة مستندات محمية بحقوق نشر.
❌ تنفيذ أوامر نظام (مثل حذف ملفات).
❌ تقديم استشارات طبية أو قانونية.
